# 🎨 GRIDS & PICS

一个简单的网格绘制与图片处理工具
- 基于Python的Pillow库

---

## 🚀 快速开始

### 系统要求
- Windows 操作系统
- 网络连接（首次使用可能需安装依赖库）

### 启动方式
双击运行 `GRIDS&PICS.bat` 文件即可启动程序

---

## ⚠️ 重要提醒

### 性能注意事项
🚨 **绘制网格时请勿使用过大的数值！** (建议 < 10,000)

### 精度说明
📐 锚点裁剪倍率无法整除时可能导致轻微误差

---

## 🎮 使用指南

### 单位系统
- **基础单位**: Pixels (像素)
- **默认值**: 按 `Enter` 键即可沿用默认设置

### 操作流程
1. 完成功能操作后输入 `/` 可重新开始当前功能
2. 输入其他任意键退出程序
3. 直接关闭窗口可能导致日志未保存

---

## 🔧 高级功能

### 调试模式
在主菜单界面输入 `#DEBUG#` 可开启日志调试模式，获取详细的运行时信息。
